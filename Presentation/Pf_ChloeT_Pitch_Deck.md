# SLIDE 3 — Project Showcase

## Featured Project: The Hunter — Automated Email Phishing Defense System

**Course:** ITAI-2376 Deep Learning (Capstone)
**🔗 [View on GitHub](https://github.com/Chloe-Tu2)**

---

### The Problem It Solves

> Email phishing is responsible for **91% of cyberattacks**. Manual review is slow, inconsistent, and doesn't scale. Organizations need autonomous, always-on threat detection.

---

### My Approach

```
Email Input
    │
    ▼
Agent 1 — The Analyst
    BiLSTM + Random Forest Ensemble
    Computes risk score (0–100)
    │
    ▼
Agent 2 — The Arbiter  [AGENTIC: Auto-escalates borderline cases]
    Quarantine / Flag / Clear decision
    │
    ▼
Agent 3 — The Archivist  [AGENTIC: Cross-session memory]
    Tracks repeat senders → increases threat level
    │
    ▼
React Web Interface — Real-time results dashboard
```

---

### Quantified Results & Impact

| Metric | Value |
|---|---|
| **Detection Accuracy** | **97%+** on Alam Phishing Dataset (~11,000 emails) |
| **Autonomous Escalation** | Agent 2 escalates borderline cases without human input |
| **Cross-Session Memory** | Agent 3 identifies repeat offenders across multiple sessions |
| **Full Stack** | React frontend + Python backend + SQLite persistence |
| **Zero Manual Intervention** | End-to-end autonomous pipeline |
