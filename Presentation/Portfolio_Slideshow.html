🏆 Most Significant Project 1 · ITAI-2376 Deep Learning Capstone

The Hunter — Automated Email Phishing Defense

A fully autonomous 3-agent AI cybersecurity pipeline achieving 97%+ phishing detection accuracy with cross-session threat memory.

Three-Agent Architecture

AGENT 1 — Scanner & Feature Extractor • Reads headers, body, URLs, sender domain • Extracts 15+ features per email • Passes feature vector → Agent 2 ↓ AGENT 2 — Classifier & Escalation Engine • BiLSTM + Random Forest Ensemble • ≥0.85 → Auto-quarantine (no human) • 0.50–0.84 → Escalate to analyst • <0.50 → Clear to inbox ↓ AGENT 3 — Repeat Offender Memory • Persistent SQLite threat database • Survives session restarts • Elevates score for known bad domains

Agentic Behavior 1

Agent 2 autonomously escalates borderline cases (conf. 0.50–0.84) — no hardcoded rules, the agent decides independently based on model confidence.

Agentic Behavior 2

Agent 3's SQLite memory persists across sessions — repeat offenders are recognized and auto-elevated even after program restarts.

Dataset

Trained on Alam Phishing Dataset — 18,000+ labeled phishing & legitimate emails. Ensemble of BiLSTM (sequential) + Random Forest (structured features).

🥈 Most Significant Project 2 · ITAI-1378 Computer Vision Capstone

Workplace Safety PPE Detection System

Real-time YOLOv8 system detecting 10 PPE categories at 66 FPS — enabling automated safety compliance monitoring on live camera feeds.

Detection Pipeline

Live Camera / Video Feed ↓ PREPROCESSING • Normalize frames to 640×640 • Mosaic augmentation (training) • Color jitter + scale transforms ↓ YOLOv8 CUSTOM-TRAINED MODEL • Backbone: CSPDarknet + C2f • Transfer learning from COCO • 100 epochs · Batch 16 · 640px ↓ 10-CLASS OUTPUT Hardhat ✓ | No-Hardhat ✗ Safety Vest ✓ | No-Vest ✗ Gloves ✓ | No-Gloves ✗ Safety Boots ✓ | No-Boots ✗ Person (worker) | Machinery
